# Macher Bari (React)

CS395 final project — bengali restaurant landing page.

Originally built in plain html/css/js, rewrote it in react + tailwind for the assignment.

### run it

```
npm install
npm run dev
```

### build for prod

```
npm run build
```

### stuff that works
- horizontal scroll on desktop (yes its weird, copied the trick from my old version)
- shopping cart (saves to localStorage)
- mobile responsive (hopefully)

### stuff that doesnt
- the gallery scroll buttons are a bit janky
- contact form just alerts, doesnt actually send anything
